# WikiAPI 📚 

## 🧾 Descrição

Esta API entrega um molde para uma enciclopédia virtual. Permite ler, criar, editar e deletar artigos.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C# (.NET 8)
- **Framework:** ASP.NET Core - Minimal API
- **ORM:** Entity Framework Core
- **Banco de Dados:** SQLite
- **Front-end:** JavaScript, HTML e CSS
- **Versionamento:** Git + GitHub

---

## 🚀 Como Executar o Projeto

### Pré-requisitos

- [.NET SDK 8.0+](https://dotnet.microsoft.com/en-us/download)
- [SQLite](https://www.sqlite.org/download.html) (ou algum cliente de visualização como DB Browser for SQLite)
- Git instalado

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/Neezic/WikiAPI

# 2. Acesse a pasta do projeto
cd WikiAPI

# 3. Restaure os pacotes
dotnet restore

# 4. Execute a aplicação
dotnet run
